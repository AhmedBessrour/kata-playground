# TypeScript Jest boilerplate

This boilerplate uses TypeScript and Jest as testing framework.

## Installing dependencies

```bash
# Get Yarn
npm install -g yarn

# Install dependencies
yarn install
```

## Running tests

```bash
# Run tests once
yarn test

# Run tests in watch mode
yarn test:watch

# Run test coverage
yarn coverage

# Run lint
yarn lint

# Run compile
yarn compile
```
